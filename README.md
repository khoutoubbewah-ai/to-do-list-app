# Daily Quest Board

Daily Quest Board is a beginner-friendly JavaScript mini project that turns a to-do list into a small storytelling experience.

## Included Files

- `index.html`
- `style.css`
- `script.js`
- `tasks.json`
- `project_overview.txt`
- `README.txt`
- `assets/quest-board-illustration.svg`

## Key Features

- Dynamic DOM rendering for tasks
- Click, keyboard, and form event handling
- Async task loading with `fetch()` and `async/await`
- JSON data from a local file
- Progress tracking, filtering, and day selection
- Smart next-task recommendation based on priority and time
- Persistent tasks with `localStorage`
- Reward points, badges, and daily streak tracking
- Animated UI with a clear visual theme

## Run

Open with a local server such as VS Code Live Server for the `fetch("tasks.json")` request to load normally.
